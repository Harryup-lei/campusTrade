一、项目概述

1.1 项目背景

随着高校学生群体消费观念转变，闲置物品流转需求日益增长。本项目面向校园场景，构建一个安全、便捷、可信任的二手交易与资源共享平台，覆盖商品买卖、求购匹配、免费赠送、信用评价、资讯公告等核心业务，并提供完善的后台管理能力。

1.2 建设目标

目标维度 说明
业务目标 实现校园内 C2C 二手交易闭环，降低闲置浪费
技术目标 采用主流前后端分离架构，便于扩展与维护
安全目标 JWT 鉴权 + Spring Security，保障用户与交易安全
体验目标 用户端简洁易用，管理端数据可视化、运营高效

五、安全与鉴权

5.1 认证流程

  1. 用户通过 /api/auth/login 提交账号密码
  2. Spring Security 校验通过后生成 JWT Token（有效期 24 小时）
  3. 前端将 Token 存入 Pinia，后续请求携带 Authorization: Bearer <token>
  4. JwtAuthenticationFilter 拦截并解析 Token，注入安全上下文

5.2 权限控制

  • 普通用户（USER）：浏览、发布、交易、评论
  • 卖家（SELLER）：同用户，可管理自己的商品
  • 管理员（ADMIN）：访问管理端，执行审核与运营操作
  • 管理端路由通过 meta.requiresAuth 守卫，后端敏感接口使用 @PreAuthorize

5.3 其他安全措施

  • 密码 BCrypt 加密存储
  • 文件上传大小限制（10MB）
  • 逻辑删除（MyBatis-Plus deleted 字段）
  • 操作日志记录管理员行为
